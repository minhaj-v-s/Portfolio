Alumicet - Website for ICET Alumni
Alumicet is a platform connecting ICET alumni with current students to build a strong, supportive community. Alumni share career resources, success stories, and industry insights to enhance student career prospects and inspire motivation. The website also keeps the community updated on college events. Built with Flutter and Node.js, Alumicet leverages Firebase for storage and scalability, aiming to strengthen the college's placement opportunities and foster a sense of belonging among the ICET community.
Safelink - Novel IoT based smart car safety system with airbag trigger
Developed an IoT based system which sends rescue messages from cars when the passenger meets with an accident. Node MCU microcontroller coupled with GPS sensor enabled to precisely locate and avail help. The website which is to be deployed at the hospital end was developed using HTML and CSS in the frontend and python Django in the backend. An additional feature for sending accident prone region alerts was also incorporated into the system through an app built using MIT app inventor
